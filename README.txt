Инструкция по работе с макросом.

1. Открыть деталь (листовую или обычную) (деталь должна быть сохранена).
2. Выделить поверхность относительно которой будет построена развертка.
3. Запустить макрос.

Итог.
В папке, где находится деталь, появится dxf файл с контуром развертки.
Размеры развертки зависят от параметров настроенных в компас-3D.
