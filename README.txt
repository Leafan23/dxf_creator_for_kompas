Особенности работы.
1. Деталь для которой делается dxf должна быть сохранена.
2. Толщина высчитывается для листовой детали исход из свойства толщины.
3. Толщина не листовой детали высчитывается исходя из габарита относительно основной системы координат. Если основная плоскость находится не параллельно относительно плоскостей основной системы координат, то толщина расчитается не корректно.

Инструкция по работе с макросом.
1. Открыть деталь (листовую или обычную).
2. Выделить поверхность относительно которой будет построена развертка. Можно не выделять поверхность, тогда будет выделена самая большая, плоская поверхность.
3. Запустить макрос.

Итог.
В папке, где находится деталь, появится dxf файл с контуром развертки.
Название будет состоять из имени файла 3D-модели, исполнения для которого делается развертка, толщины детали
Размеры развертки зависят от параметров настроенных в компас-3D.
